Bienvenido a la API de animales.

Endpoints

Consumir todas las mascotas disponibles

Consultar al endpoint: "/api/mascotas"

Retorna un JSON con todas las mascotas.

Filtrar mascotas por ID

Consultar al endpoint: "/api/mascotas/:id"

Retorna una mascota en forma JSON.

Filtrar mascotas por Name

Consultar al endpoint: "/api/mascotas/name/:name"

Retorna un Array de mascotas.
